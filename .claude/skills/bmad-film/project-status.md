# Project Status Skill

**Command**: `/project-status`
**Description**: View current project status and progress

---

## Skill Instructions

When user invokes `/project-status`:

1. **Read** `film-project-context.md`
2. **Generate** comprehensive status report
3. **Display** progress, next steps, and recommendations

```markdown
# Project Status: [Project Title]

**Phase**: [Current Phase]
**Progress**: [X]% complete
**Last Updated**: [Date]

## Overview
[Project logline and type]

## Current Status
**Active Workflows**: [List]
**Completed Milestones**: [List]
**Upcoming Milestones**: [List]

## Production Details
**Budget**: $[spent] / $[total]
**Schedule**: [status vs. plan]
**Shoot Days**: [X of Y complete]

## Next Actions
1. [Action 1]
2. [Action 2]

## Recommended Next Command
**`/[next-command]`** - [Why this is next]

---
*Run `/bmad-film-help` for more information*
```
