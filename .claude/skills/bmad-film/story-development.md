## Error Handling

**If user's concept is too vague**:
- Story Architect conducts deeper excavation
- More probing questions to uncover core
- May need multiple rounds of questioning

**If characters feel like plot devices**:
- Character Designer digs deeper into psychology
- Focus on ordinary normalcy before disruption
- Establish want vs. need more clearly

**If screenplay scenes feel static**:
- Screenplay Developer applies "Are you sure?" more rigorously
- Check every scene for goal-obstacle-outcome-turn
- May need to cut or combine scenes

**If dialogue sounds unnatural**:
- Dialogue Agent focuses on character voice authenticity
- Read aloud test
- More attention to regional/cultural patterns

**If agents' outputs don't cohere**:
- Master Agent facilitates integration session
- Identifies conflicts and synthesizes solutions
- May need one agent to adjust based on another's work
