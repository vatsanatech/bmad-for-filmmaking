# Quick Edit Skill

**Command**: `/quick-edit`
**Workflow**: `quick-edit.yaml`
**Description**: Fast-turnaround editing and delivery

---

## Skill Instructions

When user invokes `/quick-edit`:

1. **Load agents**: Editor (lead), Post-Production Supervisor, Producer
2. **Load workflow**: `.bmad-film/workflows/post-production/quick-edit.yaml`
3. **Verify prerequisites**: Check that production is complete
4. **Execute workflow**:
   - Organize and ingest footage
   - Review and select takes
   - Create first assembly
   - Refine cut based on feedback
   - Add sound and music
   - Color correction
   - Add graphics
   - Export deliverables

5. **Outputs**: Final video ready for delivery
6. **Update context** and mark project complete
