## üéØ Key Features

### **Genre-Trained Agents**
8 specialist agents trained on Bollywood masters:
- **Thriller**: Anurag Kashyap, Sriram Raghavan style
- **Romance**: Karan Johar, Imtiaz Ali style
- **Horror**: Ram Gopal Varma style
- **Comedy**: Rajkumar Hirani style
- **Action**: Rohit Shetty, Siddharth Anand style
- **Drama**: Zoya Akhtar, Shoojit Sircar style
- **Crime**: Vishal Bhardwaj style
- **Social**: Shyam Benegal style

### **Two-Agent Screenplay Model**
Industry-authentic approach:
- **Agent 1**: Screenplay Structure Writer (scenes, action, structure)
- **Agent 2**: Dialogue Writer/Samvad Lekhak (impactful dialogue)
- Credits: "Screenplay by X, Dialogues by Y" (like real Bollywood)

### **Language/Dialect Control**
- **Default**: Simple Bollywood Hindi (60-70% Hindi, 30-40% English)
- **Options**: Pure Hindi, Hinglish, Haryanvi, Bihari, Punjabi, Marathi, etc.
- Character-specific application (urban character = more English, rural = more Hindi)

### **Feedback Loops**
Review and iterate at each step:
```
"The dialogue in Scene 5 is too formal, make it natural"
"Make Ananya's backstory show more trauma"
"Add 3 more close-ups in the interrogation scene"
```

## üí° Pro Tips

1. **Be specific in Step 1**: Include theme, tone, must-have elements
2. **Review carefully**: Each step builds on previous ones
3. **Use feedback**: Don't settle for "good enough" - iterate!
4. **Choose dialect wisely**: Match character backgrounds
5. **Check shot coverage**: Ensure enough shots for editing

## ‚ùì FAQ

**Q: Do I need filmmaking experience?**
A: No! The system guides you through each step with industry standards.

**Q: Can I change earlier steps?**
A: Yes, but you'll need to regenerate subsequent steps for consistency.

**Q: How long does it take?**
A: Full workflow: 1.5-3 hours (depending on project length and iterations)

**Q: Can I edit the outputs manually?**
A: Yes, all outputs are markdown files you can edit in any text editor.

**Q: What languages are supported?**
A: Hindi (default), Hinglish, Pure Hindi, and regional dialects (Haryanvi, Punjabi, Gujarati, Marathi, Bihari/Bhojpuri, etc.)

**Q: Can I use this for feature films?**
A: Currently optimized for short films (5-20 min). Feature film support planned.
