## ğŸ“‹ Question-Based Workflow System

Each workflow step has **3 phases**:

### **Phase 1: Question Phase** (Gather Vision)
- Agent asks 10-20 targeted creative questions
- Questions based on:
  - User's initial concept
  - Project scale (short/feature/series)
  - Genre conventions
  - Filmmaking best practices

- Agent asks follow-up questions
- Iterative refinement until vision is clear

### **Phase 3: Creation Phase** (Execute)
- Agent creates output based on complete understanding
- Output is unique, customized, non-generic
- User reviews and approves

### **STEP 2: Character Bible (Interactive)**

**Input**: Approved story synopsis

**Question Phase** (Character Agent asks):

**A. Protagonist Deep Dive** (8 questions):
1. "Protagonist's biggest fear? (Not external danger, internal)"
2. "What shaped them? Key backstory event?"
3. "Flaw that holds them back?"
4. "How do they speak? (Formal, casual, witty, terse)"
5. "Physical description: Any specific look you imagine?"
6. "How do they change by end? What do they learn?"
7. "What makes them relatable despite flaws?"
8. "Any actor/persona you imagine for this role?"

**B. Supporting Characters** (5 questions per major character):
9. "For [Character X]: What's their relationship to protagonist?"
10. "What do they want from protagonist?"
11. "Do they help or hinder protagonist?"
12. "Any specific traits or quirks?"
13. "How much depth do they need? (Brief/moderate/full)"

**C. Character Dynamics** (5 questions):
14. "Key relationships: Which are most important?"
15. "Any romantic subplot? How deep?"
16. "Antagonist: Clear villain or morally complex?"
17. "Who has the best dialogue potential?"
18. "Any character inspired by real people you know?"

**Agent then**:
- Creates detailed profiles based on answers
- Presents character descriptions
- User refines
- Agent finalizes character bible

### **STEP 3: Screenplay (Interactive)**

**Input**: Approved character bible + story

**Question Phase** (Screenplay Agent asks):

**A. Screenplay Style** (6 questions):
1. "Dialogue-heavy or visual storytelling?"
2. "Scene length preference: Quick cuts or longer takes?"
3. "Dialogue style: Witty/poetic/realistic/terse?"
4. "Subtext: Subtle undertones or direct?"
5. "Opening scene: What should it establish?"
6. "Closing scene: What should final image be?"

**B. Language & Voice** (4 questions):
7. "Hindi-English ratio default okay? (60-40) Or adjust?"
8. "Any specific dialect? (Haryanvi, Punjabi, etc.)"
9. "Formal or casual language overall?"
10. "Any memorable lines/quotes you want included?"

**C. Scene Preferences** (5 questions):
11. "Any specific scenes you imagine clearly?"
12. "Action sequences: How many? How intense?"
13. "Emotional beats: Where should audience cry/cheer?"
14. "Montages needed? (Training, time passage, etc.)"
15. "Flashbacks: Use them? How many?"

**Agent then**:
- Creates screenplay structure
- Presents key scenes for review
- User refines dialogue/beats
- Agent finalizes complete screenplay

### **STEP 4: Shot Breakdown (Interactive)**

**Input**: Approved screenplay

- User refines

## ğŸ¯ Scale-Adaptive Question System

Questions adjust based on **project scale**:

### **30-Second Microdrama**:
- 5-7 questions per step (focused, essential)
- Single emotion focus
- One location preferred
- Minimal characters

- 10-12 questions per step
- Core emotional journey
- 2-3 locations
- 3-5 characters

### **15-Minute Short**:
- 15-18 questions per step
- Character arc development
- Multiple locations
- 5-8 characters

### **90-Minute Feature**:
- 25-30 questions per step
- Subplots, complex arcs
- World-building
- 10-15 characters

### **Web Series (Season)**:
- 35-40 questions per step
- Season arc planning
- Episode structure
- Character evolution over time
- Cliffhangers and hooks

### **Sequel/Franchise**:
- Additional questions about:
  - Previous film continuity
  - Character growth from last film
  - New vs returning elements
  - Expanded world-building

## ğŸ¨ Unique Story Generation Methods

### **Method 1: Constraint-Based Creativity**
- User defines what they DON'T want
- Agent explores uncharted territory
- Result: Fresh takes on familiar genres

### **Method 2: Personal Connection**
- Ask: "Any personal experience inspiring this?"
- Ground story in authentic emotion
- Result: Emotionally resonant, not generic

### **Method 3: Reference Mixing**
- User gives 2-3 reference films
- Agent extracts different elements from each
- Combines in novel way
- Result: Familiar yet fresh

### **Method 4: Subversion Prompts**
- Ask: "What would audience EXPECT here?"
- Agent deliberately subverts expectation
- Result: Surprising, non-predictable

### **Method 5: Character-First**
- Start with deeply developed character
- Story grows organically from character truth
- Result: Character-driven, not plot-mechanical
