## üé¨ STEP 1: Story Development Questions

### **Question Framework: Story Co-Writing**

**Purpose**: Understand user's vision deeply before generating story

**Question Categories**: 5 categories, 20 questions total

---

### **A. CORE VISION** (Mandatory - 5 questions)

**Q1. Emotional Core**
```
What's the ONE emotion you want audience to feel MOST?

Options to consider:
- Suspense/tension (thriller)
- Joy/laughter (comedy)
- Romance/longing (romance)
- Fear/dread (horror)
- Anger/injustice (social)
- Awe/wonder (adventure)
- Sadness/catharsis (drama)

Your answer shapes everything else!
```

**Q2. Unique Hook**
```
What makes THIS story unique?
Why does THIS story need to be told?

(What's fresh about it? Personal connection? Unexplored angle?)
```

**Q3. Target Audience**
```
Who is this film for?

Consider:
- Age group? (Kids, teens, adults, all)
- Sensibility? (Mass, niche, festival, commercial)
- Cultural context? (Pan-India, regional, global)
```

**Q4. Reference Films**
```
Any films that capture the TONE you want?

(This helps me understand your taste and vision)

Examples:
- "Like Andhadhun but darker"
- "Mix of 3 Idiots humor + Taare Zameen Par emotion"
- "Gangs of Wasseypur realism"
```

**Q5. Lasting Impact**
```
What should audience remember a WEEK after watching?

- A specific scene/moment?
- An emotional feeling?
- A character?
- A dialogue line?
- A message/theme?
```

---

### **B. STORY FUNDAMENTALS** (Mandatory - 5 questions)

**Q6. Protagonist**
```
Who is your protagonist? (Basic idea)

Consider:
- Age, gender, background?
- Occupation?
- Personality trait (brief)?
- Why are THEY the protagonist?
```

**Q7. Protagonist's Want**
```
What does protagonist want desperately?

(Not what they NEED, but what drives them in the story)

External goal examples:
- Solve a mystery
- Win someone's love
- Escape a situation
- Achieve revenge
- Save someone
```

**Q8. Central Conflict**
```
What's stopping them?

- External obstacle? (Villain, system, circumstances)
- Internal obstacle? (Fear, flaw, doubt)
- Both?
```

**Q9. World/Atmosphere**
```
What's the WORLD like?

Tone:
- Gritty realistic or stylized?
- Dark/noir or bright/hopeful?
- Urban or rural?
- Contemporary, period, or future?

Atmosphere:
- Claustrophobic or expansive?
- Grounded or heightened?
```

**Q10. Ending Direction**
```
How should it END? (General direction, not details)

- Triumph? (Protagonist wins)
- Tragedy? (Protagonist loses/dies)
- Bittersweet? (Wins but loses something)
- Ambiguous? (Open to interpretation)
- Twist? (Reframes everything)
```

---

### **C. GENRE & STYLE** (Conditional - 5 questions)

**Q11. Genre Mix**
```
Genre: Pure or mixed?

If PURE:
- Pure thriller, pure romance, pure comedy, etc.

If MIXED:
- Primary genre (60-70%)?
- Secondary elements (20-30%)?

Example: "Romantic thriller" = Thriller primary, Romance secondary
```

**Q12. Pacing**
```
Pacing preference?

- Fast and tight (non-stop, no breathing room)
- Moderate (balanced action and emotion)
- Slow-burn (building tension, patient)
- Variable (slow start, explosive climax)
```

**Q13. Tone Specifics**
```
Tone refinement:

Dark ‚Üî Light?
- How dark is dark? (Disturbing or moody?)
- How light is light? (Playful or heartwarming?)

Realistic ‚Üî Stylized?
- Grounded and natural?
- Heightened/theatrical?
- Magical realism?
```

**Q14. Twist/Reveal**
```
Story structure approach?

- Shocking twist at end? (Sixth Sense style)
- Emotional revelation? (Character-driven turn)
- Gradual unfolding? (No single twist, slow reveal)
- Straightforward? (No tricks, pure emotion)
```

**Q15. Setting Detail**
```
Setting specifics:

- Location? (City/town/village, which one?)
- Time period? (If not contemporary, when?)
- Cultural context? (Specific community/class?)
- Any specific locations you MUST see?
```

---

### **D. MUST-HAVE ELEMENTS** (Optional - 5 questions)

**Q16. Specific Scenes**
```
Any specific scenes you MUST see in the film?

Examples:
- "Confrontation between protagonist and antagonist"
- "Flashback revealing protagonist's trauma"
- "Montage showing time passing"
- "Final speech/monologue"

List any you're imagining!
```

**Q17. Elements to AVOID**
```
Any clich√©s, tropes, or elements you HATE?

Examples:
- "No damsel in distress"
- "No convenient coincidences"
- "No villain monologuing"
- "No unrealistic fight scenes"

This helps me steer away from what you dislike!
```

**Q18. Character Types**
```
Any specific character TYPES you want?

Examples:
- Mentor figure
- Comic relief sidekick
- Rival/competitor
- Tragic backstory character
- Mysterious stranger
- Voice of reason
- Chaos agent

(Not full characters yet - just archetypes)
```

**Q19. Theme/Message**
```
Any theme or message you want to convey?

Examples:
- "Revenge is hollow"
- "System vs individual"
- "Love transcends circumstances"
- "Truth always emerges"

Or no specific message - pure entertainment?
```

**Q20. Personal Inspiration**
```
(Optional, but powerful)

Is this story inspired by:
- Personal experience?
- Real events?
- Someone you know?
- Something that angered/moved you?

(Stories rooted in truth feel more authentic)
```

---

### **Question Delivery Strategy**:

1. **First Batch** (Q1-Q5): Core vision - always ask
2. **Second Batch** (Q6-Q10): Story fundamentals - always ask
3. **Third Batch** (Q11-Q15): Genre/style - conditional based on answers
4. **Fourth Batch** (Q16-Q20): Specific elements - optional, user can skip

**Total Question Time**: 10-15 minutes for thorough answers

## üé≠ STEP 2: Character Bible Questions

### **Question Framework: Character Development**

**Purpose**: Build three-dimensional, non-generic characters

**Question Categories**: 3 categories, 18 questions

---

### **A. PROTAGONIST DEEP DIVE** (Mandatory - 8 questions)

**Q1. Internal Fear**
```
What is protagonist's BIGGEST FEAR?

(Not external danger like "being killed" - internal psychological fear)

Examples:
- Being alone/abandoned
- Being exposed as fraud
- Losing control
- Repeating parent's mistakes
- Never mattering/being remembered
```

**Q2. Formative Event**
```
What SHAPED them into who they are?

Key backstory event that explains:
- Their worldview
- Their fear/wound
- Their defense mechanisms
- Their desires

(One specific event, not entire history)
```

**Q3. Fatal Flaw**
```
What FLAW holds them back?

Character flaw that:
- Creates their own problems
- Needs to be overcome by end
- Is relatable (we all have it)

Examples:
- Pride/arrogance
- Inability to trust
- Avoiding confrontation
- Obsessive control
- Self-destructive patterns
```

**Q4. Speaking Voice**
```
How do they SPEAK?

Voice characteristics:
- Formal or casual?
- Verbose or terse?
- Witty or serious?
- Direct or evasive?
- Confident or hesitant?

Any speech patterns or verbal tics?
```

**Q5. Physical Presence**
```
Physical description - any specifics you imagine?

Not full casting, but:
- Age range?
- Build/stature?
- Any distinctive features?
- How do they carry themselves?
- Any specific look you're picturing?

(Helps with casting direction and visual description)
```

**Q6. Character Arc**
```
How do they CHANGE by the end?

Journey:
- Who are they at START?
- What does story DO to them?
- Who are they at END?
- What have they LEARNED?

(Internal transformation, not just external events)
```

**Q7. Relatability Factor**
```
What makes them RELATABLE despite flaws?

Why should audience root for them?
- Universal struggle?
- Vulnerable moments?
- Good intentions gone wrong?
- Humor/self-awareness?
```

**Q8. Casting Inspiration**
```
(Optional but helpful)

Any actor/persona you imagine in this role?

Not for actual casting, but helps me understand:
- The energy you imagine
- The screen presence
- The emotional range needed

Example: "Like young Irrfan - subtle, intelligent"
```

---

### **B. SUPPORTING CHARACTERS** (Per character - 5 questions)

**For EACH major supporting character, ask:**

**Q9. Relationship to Protagonist**
```
What's their relationship to protagonist?

- Family? (Parent, sibling, child)
- Friend? (Old friend, new friend)
- Love interest?
- Rival/enemy?
- Mentor/guide?
- Obstacle/antagonist?
```

**Q10. Their Want**
```
What do THEY want from protagonist?

Not what protagonist wants - what does THIS character want?
- Help?
- Approval?
- To protect them?
- To use them?
- To stop them?
```

**Q11. Help or Hinder**
```
Do they HELP or HINDER protagonist's goal?

- Actively help
- Want to help but complicate
- Neutral observer
- Unwitting obstacle
- Active antagonist
```

**Q12. Defining Trait**
```
What's their most MEMORABLE trait or quirk?

Something that makes them distinct:
- Speech pattern
- Physical characteristic
- Behavioral tic
- Personality extreme
- Surprising contradiction
```

**Q13. Depth Level**
```
How much depth does THIS character need?

- FULL depth (like protagonist - full arc, backstory)
- MODERATE depth (clear motivations, some arc)
- BRIEF (functional role, memorable but simple)

(Depends on screen time and story importance)
```

---

### **C. CHARACTER DYNAMICS** (Relationship - 5 questions)

**Q14. Most Important Relationships**
```
Which character relationships are MOST important to the story?

Rank top 3:
1. [Example: Protagonist + Mother]
2. [Example: Protagonist + Antagonist]
3. [Example: Protagonist + Love Interest]

These will get the most development time.
```

**Q15. Romantic Element**
```
Is there a ROMANTIC subplot?

If YES:
- How important? (Central or background?)
- How much screen time?
- Does romance COMPLICATE or HELP protagonist's goal?
- Should it feel earned or doomed?

If NO:
- Confirmed no romance, or open to it if organic?
```

**Q16. Antagonist Nature**
```
About the antagonist/obstacle:

- Clear VILLAIN (evil, must be stopped)?
- MORALLY COMPLEX (grey area, understandable)?
- SYSTEM/CIRCUMSTANCE (not a person)?
- INTERNAL (protagonist's own flaw)?
- NONE (pure character study)?
```

**Q17. Dialogue Star**
```
Which character has the BEST dialogue potential?

Who should get the memorable lines?
- Protagonist (obviously)
- Antagonist (scene-stealing villain)
- Sidekick (comic relief)
- Mentor (wisdom)
- Multiple (ensemble)
```

**Q18. Real-Life Inspiration**
```
(Optional)

Any characters inspired by real people?

- Someone you know?
- Historical figure?
- Public personality?
- Composite of multiple people?

(Real inspiration adds authenticity)
```

---

### **Question Delivery Strategy**:

1. **Protagonist Questions** (Q1-Q8): Always ask all
2. **Supporting Characters** (Q9-Q13): Ask for top 2-3 characters
3. **Dynamics** (Q14-Q18): Always ask

**Total Question Time**: 15-20 minutes

## üìù STEP 3: Screenplay Questions

### **Question Framework: Screenplay Crafting**

**Purpose**: Determine screenplay style and approach

**Question Categories**: 3 categories, 15 questions

---

### **A. SCREENPLAY STYLE** (Mandatory - 6 questions)

**Q1. Dialogue vs Visual**
```
Is this a DIALOGUE-heavy or VISUAL storytelling film?

Dialogue-heavy:
- Characters talk, reveal through conversation
- Tarantino, Linklater style
- Example: 12 Angry Men

Visual storytelling:
- Minimal dialogue, show don't tell
- Actions, expressions, cinematography
- Example: Mad Max Fury Road

Balanced:
- Both equally important
```

**Q2. Scene Length**
```
Scene length and cutting preference?

- QUICK CUTS (rapid, energetic, 1-2 min scenes)
- LONG TAKES (patient, immersive, 3-5 min scenes)
- MIXED (varies based on emotional need)
```

**Q3. Dialogue Style**
```
What kind of DIALOGUE do you imagine?

- WITTY/SHARP (clever, quotable, Tarantino-esque)
- POETIC/LYRICAL (beautiful, metaphorical, Gulzar-style)
- REALISTIC/NATURAL (how people actually talk, mumblecore)
- TERSE/MINIMAL (Eastwood style, says lot with little)
- THEATRICAL (heightened, dramatic, stage-like)
```

**Q4. Subtext Level**
```
How much SUBTEXT?

Subtext = what's unsaid, hidden meanings

- HEAVY subtext (characters never say what they mean)
- MODERATE subtext (some hidden meanings)
- DIRECT (characters say what they feel, no games)
```

**Q5. Opening Scene**
```
How should film OPEN?

- Action/intensity (grab immediately)
- Quiet/atmospheric (slow build)
- Mystery (raise questions)
- Character introduction (meet protagonist)
- Dramatic event (inciting incident)

Any specific opening image you imagine?
```

**Q6. Closing Scene**
```
How should film END?

Final image audience sees:
- Character's face (emotion)
- Symbolic image (thematic)
- Action moment (dynamic)
- Quiet reflection (contemplative)
- Ambiguous visual (interpretation)

What should final FEELING be?
```

---

### **B. LANGUAGE & VOICE** (Mandatory - 4 questions)

**Q7. Hindi-English Balance**
```
Language ratio preference?

Default: 60-70% Hindi, 30-40% English (Simple Bollywood Hindi)

Adjust?
- More Hindi (70-80%)
- More English (urban, 40-50%)
- Pure Hindi (90%+, formal)
- Pure Hinglish (50-50, youth)
```

**Q8. Dialect/Regional Flavor**
```
Any DIALECT or regional language influence?

Options:
- No dialect (neutral Bollywood Hindi)
- Haryanvi (rural Haryana)
- Punjabi-influenced (energetic, expressive)
- Bihari/Bhojpuri (earthy, colorful)
- Marathi-influenced (Mumbai local)
- Bengali-influenced (literary, poetic)
- Southern-influenced (Tamil/Telugu mixing)

Per character or overall tone?
```

**Q9. Formality Level**
```
Overall language formality?

- CASUAL (tum, yaar, bhai)
- FORMAL (aap, sir, respect terms)
- MIXED (depends on character dynamics)

(This affects intimacy and relatability)
```

**Q10. Memorable Lines**
```
Any specific LINES or quotes you want?

If you're imagining any:
- Key dialogue exchange?
- Character catchphrase?
- Emotional monologue?
- Memorable one-liner?

(Not required, but if you have ideas, share!)
```

---

### **C. SCENE PREFERENCES** (Optional - 5 questions)

**Q11. Specific Scenes You Imagine**
```
Any scenes you can ALREADY visualize?

Describe what you see:
- Character doing what?
- Location?
- Mood/feeling?
- Why this scene is important?

(Even vague descriptions help!)
```

**Q12. Action Sequences**
```
(If action present in story)

How many action sequences?
How intense?
- Realistic or stylized?
- Short bursts or extended?
- Hand-to-hand, vehicular, gunfights?

Style references?
```

**Q13. Emotional Beats**
```
Where should audience CRY, CHEER, or GASP?

Mark the emotional high points:
- Cry: Touching/tragic moment
- Cheer: Triumph/satisfaction moment
- Gasp: Shock/surprise moment

(Helps me structure emotional journey)
```

**Q14. Montages**
```
Any MONTAGES needed?

Common montages:
- Training/preparation
- Time passing
- Falling in love
- Building something
- Search/investigation
- Decline/deterioration

How many? What do they show?
```

**Q15. Flashbacks**
```
Should we use FLASHBACKS?

If YES:
- How many?
- What do they reveal?
- Chronological or scattered?
- Brief or extended?

If NO:
- Linear storytelling only?
```

---

### **Question Delivery Strategy**:

1. **Style Questions** (Q1-Q6): Always ask
2. **Language** (Q7-Q10): Always ask
3. **Specific Scenes** (Q11-Q15): Ask if user wants to specify

**Total Question Time**: 10-15 minutes

## üé• STEP 4: Shot Breakdown Questions

### **Question Framework: Visual Design**

**Purpose**: Understand visual style and production constraints

**Question Categories**: 3 categories, 15 questions

---

### **A. VISUAL STYLE** (Mandatory - 7 questions)

**Q1. Visual Tone**
```
What's the overall VISUAL TONE?

Brightness:
- BRIGHT (high key lighting, optimistic)
- DARK (low key, shadows, mysterious)
- BALANCED (natural, varied)

Color:
- COLORFUL (saturated, vibrant)
- MUTED (desaturated, subdued)
- MONOCHROMATIC (limited palette)
- NATURAL (realistic color)
```

**Q2. Camera Movement**
```
Camera movement preference?

- STATIC (locked on tripod, stable, classical)
- HANDHELD (shaky, intimate, documentary feel)
- SMOOTH (steadicam/gimbal, flowing, dreamlike)
- MIXED (varies by scene emotion)

General vibe:
- Restrained or dynamic?
```

**Q3. Shot Size Preference**
```
Shot size tendency?

- WIDE shots (environment, space, isolation)
- TIGHT shots (close-ups, faces, intimacy)
- MIXED (balanced coverage)

Bias toward one?
```

**Q4. Lighting Mood**
```
Lighting approach?

- NATURAL (realistic, window light, sun)
- DRAMATIC (high contrast, shadows, noir)
- SOFT (diffused, gentle, romantic)
- HARSH (hard light, intensity, thriller)
- MIXED (depends on scene)

Mood words that describe lighting?
```

**Q5. Camera Angles**
```
Any specific camera ANGLES you love?

Options:
- Eye level (neutral, standard)
- Low angle (power, intimidation)
- High angle (vulnerability, god's view)
- Dutch angle (disorientation, unease)
- Mix based on character/emotion

Preference or director's choice?
```

**Q6. Visual References**
```
Reference films for VISUAL STYLE?

(Different from story references - this is cinematography)

Examples:
- "Tumbbad's dark atmosphere"
- "Zindagi Na Milegi Dobara's bright colors"
- "Gangs of Wasseypur's gritty realism"
- "Padmaavat's grandeur"
```

**Q7. Must-See Shots**
```
Any SPECIFIC SHOTS you must see?

Examples:
- "Silhouette shot at sunset"
- "Overhead shot of character lying down"
- "Slow-motion rain scene"
- "Mirror reflection shot"
- "Long tracking shot through location"

(Help me prioritize your vision)
```

---

### **B. COVERAGE APPROACH** (Conditional - 4 questions)

**Q8. Coverage Philosophy**
```
Coverage priority?

MORE options:
- Multiple angles per scene
- Safety for editing
- Flexibility in post

MINIMAL:
- Specific shots only
- Intentional, planned
- Confidence in vision

(More coverage = more shooting time)
```

**Q9. Close-Up Usage**
```
Close-ups preference?

- HEAVY use (emotion, intimacy, Sanjay Leela Bhansali style)
- SELECTIVE (only key moments)
- MINIMAL (prefer wider shots)

When should we go close?
```

**Q10. Establishing Shots**
```
Establishing shots approach?

- DETAILED (long, immersive establishing)
- QUICK (brief, get to action)
- SKIP (assume audience knows location)

Importance of showing WHERE we are?
```

**Q11. Transitions**
```
Transition style?

- CUTS (standard, 90% of films)
- DISSOLVES (dreamlike, time passing)
- CREATIVE (wipes, match cuts, visual tricks)
- MIXED (varies by emotion)

Should transitions be invisible or stylish?
```

---

### **C. PRODUCTION REALITY** (Mandatory - 4 questions)

**Q12. Equipment Available**
```
What equipment do you have?

Camera:
- Smartphone
- DSLR/Mirrorless
- Cinema camera (RED, ARRI, etc.)
- Multiple cameras

This affects shot complexity!
```

**Q13. Crew Size**
```
Crew size?

- SOLO (you alone, minimal crew)
- SMALL (2-5 people)
- MEDIUM (6-15 people)
- FULL (15+ people, departments)

Affects what shots are possible.
```

**Q14. Location Constraints**
```
Location situation?

- ONE location (single room/space)
- FEW locations (2-3 places)
- MANY locations (5+ places)
- LOCATION FLEXIBLE (can shoot anywhere)

Access/permissions situation?
```

**Q15. Shooting Time**
```
Time available for shooting?

- 1 DAY (micro-budget, run-and-gun)
- 2-3 DAYS (short film typical)
- 1 WEEK (comfortable short film)
- 2+ WEEKS (feature or series)

Affects shot complexity and count.
```

---

### **Question Delivery Strategy**:

1. **Visual Style** (Q1-Q7): Always ask
2. **Coverage** (Q8-Q11): Ask if user wants input, else director's choice
3. **Production Reality** (Q12-Q15): Always ask (critical for practical planning)

**Total Question Time**: 10 minutes
