### **B. STORY FUNDAMENTALS** (Always Ask - 5 Questions)

**Q6. Protagonist**
```

**Q7. Protagonist's Want**

- Revenge lena

```

**Q8. Central Conflict**
```

```

**Q9. World/Atmosphere**
```

**Q10. Ending Direction**
```

## Scale-Adaptive Questioning

Questions adjust based on **project scale**:

### **30-Second Microdrama**:
- Ask 7-10 questions (core only)
- Focus: Single emotion, one beat
- Skip: Subplot, character depth questions

### **5-Minute Short**:
- Ask 12-15 questions (essential)
- Focus: Core journey
- Skip: Complex subplot questions

### **15-Minute Short**:
- Ask 18-20 questions (full set)
- Focus: Character arc, world
- Include: Some subplot questions

### **Feature Film (90+ min)**:
- Ask all 20 + follow-ups
- Focus: Complex arcs, subplots
- Deep dive: World, themes, subplots

### **Web Series**:
- Ask all 20 + series-specific
- Additional: Season structure, hooks
- Deep dive: Character evolution

### **Sequel**:
- Ask 15-18 (assume some known)
- Additional: Continuity, growth from previous
- Focus: What's NEW
